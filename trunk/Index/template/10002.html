品牌信息
皮鞋的日常护理可令皮鞋历久如新，更耐穿。
1 ) 一双皮鞋最好隔天 穿一次，因为皮鞋穿了整日之后汗水会鞋内产生湿气，所以回家后应将皮鞋放在阴凉通风处晾干，以防滋生细菌。
2 ) 平常穿着皮鞋后，应用鞋布或软毛擦去尘土，鞋跟与鞋身的隙缝位置，则可用尖头刷子擦除尘垢，然后放入鞋撑防止皮革变形。如没有鞋撑可放入报纸代替。
3 ) 若皮鞋弄湿了应用干布吸去皮鞋的水份，后放在阴凉通风处让皮鞋自然风干，千万不要把湿了的鞋子放在太阳下曝晒、冷气旁边或用风筒吹干，否则容易出现爆裂现象，清洁皮鞋时切忌用水洗和接触化学溶剂。
1982年，Leaveland(意大利枫叶)开始拓展亚洲市场，首先登陆香港，并在往后10年内，在亚太区先后拓展了马来西亚、日本、新加坡、韩国、以及澳大利亚在内的市场。
1990年，Leaveland(意大利枫叶)正式进军中国，成为90年北京亚运会中国体育代表团赞助单位。
2010年，中国嘉隆（厦门）商贸有限公司取得Leaveland(意大利枫叶)在中国的总运营权及所有代理权，并开始全力进军电子商务市场。
